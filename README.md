# Iluminação Pública Inteligente com IoT

Este repositório contém o código e a documentação do projeto desenvolvido para a disciplina de **Objetos Inteligentes Conectados**. O foco do sistema é a automação da iluminação pública utilizando ESP32, contribuindo para o desenvolvimento de cidades mais sustentáveis (ODS 11).

## 🛠 Tecnologias Utilizadas
- **Hardware:** ESP32, Sensor LDR (luminosidade), Sensor PIR (presença), LED.
- **Protocolo:** MQTT para comunicação de dados.
- **Simulação:** Wokwi.

## 📁 Estrutura do Projeto
- `sketch.ino`: Código-fonte principal do projeto.
- `/docs`: Documentação e figuras do sistema.

## 🚀 Como Executar
1. [Acesse a simulação do projeto no Wokwi aqui](https://wokwi.com/projects/465576684917830657)
2. O sistema iniciará a leitura dos sensores e publicará os dados no broker MQTT configurado.
